# Pöttinger Air Curtain — Mobile App UI

Interactive mobile app prototype for controlling the **Pöttinger Air Curtain**, an ISOBUS-connected agricultural accessory that protects wildlife during mowing operations.

**Pöttinger × Kunstuniversität Linz · Summer 2024 · Design by Simay Uslu · [simayy.com](https://simayy.com)**

---

## Overview
The Air Curtain unit mounts on the mower arm and directs pressurised air downward, gently displacing insects and small animals before the blades reach them.

## Features
- Mower start/stop — curtain state tied to mower operation
- Auto-start toggle — activates curtain automatically when mowing begins
- Air pressure slider — adjustable from low to full pressure
- Live animated device illustration responding to pressure setting
- Real-time CURTAIN ACTIVE / OFF status badge
- SensoSafe + ISOBUS system context

## Run Locally
```bash
npm install
npm run dev
```
